Object : 
 * Objects are basic run time entities in object oriented program
 * The objects are represented like a person, a place, a bank account,a table of data or any item than can handle by program
 * Object is a real world entity
	Example :
		class StduentDemo
		{
			void name()
			{

			}
			void Reg.No()
			{

			}
			public static void main(String[] args)
			{
				StudentDemo sd =new StudentDemo();

		}
